# Job Portal SaaS - Frontend

This is the frontend code for the Job Portal SaaS application. 

It is built using **Next.js**, **Tailwind CSS**, and **TypeScript**.

## How to Run Locally

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

*Note: You also need to run the backend server separately for the application to work completely.*
